# Finaldex API

## Pre requisitos
- Ter o nodeJS
- Ter o yarn ou npm

## Como executar o projeto no terminal

 1- Faça o clone do projeto
 2- Instale as dependências
 3- Rode o proteto

 ## Como rodar o projeto no docker

 1- Construa o container
 docker-compose build
 2- Rode a imagem
docker-compose up -d

### Comandos Extras
docker-compose stop
docker-compose start
